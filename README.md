# PixisetFavoritesSelector
A tool to copy into a new folder the photos that are selected by client in pixiset


## Download

| Platform | Link |
|----------|------|
| Mac      | [Download for Mac](https://github.com/gael-courmont/PixisetFavoritesSelector/releases/latest/download/PixiesetSelector-mac.zip) |
| Windows  | [Download for Windows](https://github.com/gael-courmont/PixisetFavoritesSelector/releases/latest/download/PixiesetSelector.exe) |


## On Mac

On Mac os you need to allow the app to run after trying to opening it once
- go to System settings -> Privacy & Security -> Allow applications from
- Allow pixisetFavoritesPicker

## USE

- Go to pixieset favorite activity on the galleri
- Select Lightroom copy list
- Copy
- Paste the file name in the tool
- Select a source and target folder
- add a client name
- Click search and export
